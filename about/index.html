关于“萌哩”

moely 由 moe 和 ly 组成，名字灵感来自于 bitly，中文音译为“萌哩”。

“萌哩番剧”是萌哩（moely）旗下的独立站点，致力于向用户推荐优质动漫番剧。本站点主要聚焦于番剧信息归类和数据同步，运营内容与萌哩主站相互独立，与萌哩其他产品无数据关联。

本站构建后的静态网页已在 GitHub 开源，您可以前往 开源仓库 了解更多细节。

网站特性

  • 自动同步：基于自动化任务定时同步互联网公开数据仓储，确保番剧数据及时更新。
  • 高级筛选：提供按年份、评分区间、标签等多个维度的精准过滤与高级筛选功能。
  • 权重分配：内置搜索引擎权重匹配算法，精准关联番剧译名、原名、制作组等关键信息。
  • 收藏列表：支持与用户中心无缝绑定，实现番剧收藏功能，收藏列表云端同步。
  • PWA支持：完整支持渐进式 Web 应用，可添加至桌面，并提供离线缓存访问能力。
